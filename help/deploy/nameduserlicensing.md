# 即將到期的序號？ 為何是指定使用者授權的時機

在現今的環境下，IT管理必須快速轉向全數位體驗。 隨著團隊瞭解如何在家中有效工作，對協助業務持續性的工具需求也日益增加。 這些挑戰會對任何大型IT部署造成影響，尤其是在管理虛擬工作環境中的使用者時。

透過指名使用者授權(NUL)授權Adobe產品，可讓IT管理員透過Admin Console輕鬆部署和管理Adobe工具，並讓使用者存取世界級的創意和生產力應用程式以及Adobe雲端服務，即使整個工作人員都位於偏遠地區。

如果您的組織未啟用NUL，且仍使用Adobe產品的序列化部署，則您的序號可能即將到期。 為了避免服務中斷，您必須在序號到期之前規劃移轉至NUL。

## 關於指名使用者授權

[具名使用者授權](https://helpx.adobe.com/tw/enterprise/using/licensing.html)是一種軟體授權模式，允許將軟體指派給個人，而非序號或裝置。 NUL透過進階的使用者身分管理，為IT管理員提供企業級安全性，並可使用Adobe Admin Console輕鬆部署和管理應用程式。

## 指名使用者授權的優點：

* 使用Admin Console輕鬆進行IT管理。

  Admin Console是由Adobe託管的集中式Web主控台，可讓IT管理員建立和管理使用者，以及授與Creative Cloud和Document Cloud應用程式及服務的存取權。 透過Admin Console管理您的使用者可讓上線和離線更輕鬆，並為IT管理員提供他們需要的工具。

* 充分利用專家服務。

  您與Adobe簽署的企業授權合約可透過電話或網路會議，為Adobe專家提供無限制的30分鐘工作階段，協助您解決任何問題。 若要預約工作階段，請前往Admin Console中的[支援]索引標籤，然後按一下[要求專家工作階段]&#x200B;**&#x200B;**。

* 大量使用者？ 沒問題。

  如果您有大量使用者，請考慮使用Admin Console的其他額外功能。 使用者同步工具可讓您自動執行建立使用者和產品指派的程式，並透過與組織Active Directory的資訊同步，讓使用者與產品指派在Admin Console中保持最新狀態。

* 體驗更快速的流程，甚至於遠端。

  我們不僅被要求更快速、更有效率地工作，還被要求從遠端完成這項工作。 進行數位轉型並非難事，而NUL功能以及與Adobe PDF服務的整合可讓企業將紙面流程轉換為全面的數位功能。 這也意味著員工可以輕鬆地在任何地方維持業務連續性，即使他們是在家工作。

* 生產力與協同合作的最佳工具。

  透過利用NUL，您的使用者將可存取Adobe Creative Cloud和Adobe Document Cloud等生產力強大廠商。 您可以執行很多操作：安全地建立並簽署檔案；快速建立自訂PDF；在平板電腦上開始創意專案，再將其放到案頭上；透過Document Cloud與團隊共用檔案；以及在Adobe XD中處理新的團隊專案。 我們的雲端服務可讓您輕鬆共用資產，並以遠端團隊身分進行共同作業。

現代工作流程挑戰需要現代的解決方案。 讓我們協助您轉換至指名使用者授權。 檢視[具名使用者授權移轉指南](https://offers.adobe.com/content/dam/offer-manager/en/na/marketing/CCE/Adobe_Named_User_Licensing_Migration_Guide.pdf)，開始進行轉換。
