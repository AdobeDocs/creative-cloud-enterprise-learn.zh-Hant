# MAX 2020 — 企業作業階段

作為企業創意人員，您需要與分散的團隊共同作業、建立可擴充的流程，並遵守公司系統和准則。 這些教學課程將協助您從企業角度瞭解2021年Creative Cloud版本中的新功能。

## 大規模運作 — 利用AEM Assets和InDesign Server的強大功能(26:54)

>[!VIDEO](https://video.tv.adobe.com/v/327112?hidetitle=true)

**說明**

您的創意人員是否花費太多時間在手動重複工作上？ 協助您的組織發揮創意專業人士的最大效益。 AEM和InDesign Server等企業系統可提供創意和生產人員將內容快速推送到目標受眾手中的方式。

在此直播錄影會議中，您會在中看到範本式工作流程的範例：
* Adobe Experience Manager (AEM) Assets是數位資產管理(DAM)解決方案，可與Adobe Creative Cloud整合，以協助DAM使用者與創意團隊合作，精簡內容建立過程中的共同作業
* Adobe InDesign Server是一種版面配置與撰寫引擎，可整合至其他系統來支援自動發佈解決方案

## 新常態(29:57)的新工具

>[!VIDEO](https://video.tv.adobe.com/v/328232?hidetitle=true)

**說明**

WFH帶來了挑戰，但也迫使創意人員及其公司嘗試新的工具和新的建立方式。 探索熟悉的工具，例如Illustrator和Photoshop在新的iPad版本中，以及在平板電腦(iPad、Microsoft Surface)和現在的iPhone上使用Fresco！

在這場錄製的即時座談中，您將瞭解如何：
* 在Fresco中使用各種筆刷和陰影技術，建立手繪行銷活動圖稿
* 在iPad上的Illustrator中建立並共用問題圖示以符合品牌化
* iPad上Fresco和Illustrator的複合圖稿，用來建立社群媒體頻道的內容，同時在iPad上使用Photoshop

## Collaboration與CC Libraries (27:58)

>[!VIDEO](https://video.tv.adobe.com/v/328199?hidetitle=true)

**說明**

有了Adobe Creative Cloud Libraries，您可以直接從最愛的Creative Cloud應用程式管理、組織和存取您的標誌、顏色等。 提高工作效率、確保創意一致性，並輕鬆與團隊保持同步。

在本教學課程中，您將學習如何：
* 在所有創意應用程式中輕鬆找到所需的資產
* 讓整個組織的溝通者能夠建立品牌上的體驗，同時仍保持控制
