# [!DNL Capture]

Adobe [!DNL Capture]將行動電話和平板電腦轉換為設計收集工具。 使用者可以建立許多不同型別的資產（圖稿的個別數位元素）。   這些資產會自動與其他案頭和行動Adobe應用程式同步。 使用者可將這些區段帶入其創意專案或輕鬆與共同作業人員分享。
