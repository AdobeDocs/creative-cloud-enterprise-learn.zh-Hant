# [!DNL Capture]

![教學課程主圖影像](../assets/Capture.jpg)

[!DNL Capture]Adobe將行動電話和平板電腦轉變為設計集合工具。使用者可以建立許多不同類型的資產 （適用于作品的個別數位元素）。   這些資產會自動與其他桌面和行動Adobe應用程式同步。 使用者可以將其匯入其創意專案或輕鬆與共同作業人員共用。
