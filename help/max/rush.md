# [!DNL Rush]

Premiere [!DNL Rush]是第一個多合一的跨裝置視訊編輯應用程式，讓線上內容的建立和共用變得前所未有的容易。 此整合式桌上型電腦與行動解決方案會自動將您的專案與編輯同步至雲端，讓您在任何裝置、任何地方都可工作。

## 建立社群媒體影片(18:11) {#tutorial1}

>[!VIDEO](https://video.tv.adobe.com/v/326900?hidetitle=true)

**描述**
使用Adobe [!DNL Stock]的視訊和音訊來講述您的故事。 Adobe [!DNL Rush]可讓您在任何裝置上工作，並讓初學者輕鬆進行專業輸出。

在本教學課程中，您將學習如何：
* 在桌上型電腦、平板電腦和手機上無縫編輯視訊
* 使用自動重新框架AI技術功能，讓主題在水準、正方形和垂直外型規格之間保持置中
* 動態圖形範本(MoGRTS)提供專業外觀及可輕鬆自訂的標題，以及三分之二
* 輕鬆匯出並直接發佈至社群媒體頻道
* 在Adobe Premiere Pro中開啟[!DNL Rush]個專案
