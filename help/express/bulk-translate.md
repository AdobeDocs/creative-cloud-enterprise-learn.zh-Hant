# 如何大量翻譯您的內容

瞭解如何自動翻譯46種不同語言的內容，以建立專案的語言變體。 您可以選取想要的語言、複製和翻譯內容，以及保留所有動畫。

>[!NOTE]
>
>在共用或下載前，請務必檢閱翻譯的準確性。
