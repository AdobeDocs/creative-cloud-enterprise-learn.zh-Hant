# 如何使用元素

瞭解如何將視覺化設計元素新增至專案，讓專案更具吸引力及吸引人。 有四種型別的元素：設計資產、背景、形狀和圖示。 每個元素型別都有數百個不同的元件可供選擇。 每個元素都經過專業設計，可隨時新增至專案。
