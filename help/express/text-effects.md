# 如何新增文字

瞭解在創意專案中新增文字的所有不同方式，包括編輯、移動和刪除文字圖層、變更字型、調整文字大小和版面、對齊文字、變更填色顏色和輪廓、新增陰影，以及使用形狀及剪下文字。 提供建議的字型以供靈感。
