# 自訂Adobe [!DNL Stock] 向量插圖中的顏色

使用精美的插圖，為任何專案增添修飾效果。 在Adobe中找到完美的向量 [!DNL Stock] ，然後使用「Adobe Illustrator」讓顏色與專案的浮動視窗相符。
