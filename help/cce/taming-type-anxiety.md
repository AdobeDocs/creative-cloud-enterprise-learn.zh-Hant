# 坦平文字特性

在Adobe Creative Cloud訂閱眾多好處中，有 20，000 多種專業設計及精選的字體可從Adobe Fonts中免費取用。 Adobe Fonts最初稱為 Typekit，透過單一授權合約提供，可供設計人員為個人或商業專案提供無限的創意使用。
