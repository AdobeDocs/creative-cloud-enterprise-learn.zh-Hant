# 與適用於團隊的Creative Cloud Libraries共用品牌化資產

瞭解如何使用適用於團隊的Creative Cloud Libraries為您的群組或團隊建立、使用和共用品牌化資產。 Creative Cloud Libraries適用於團隊的Assets儲存在中央位置，且不會繫結至個別使用者帳戶。 您可以透過![建置影像](assets/Smock_Building_18_N.png)圖示來識別團隊資料庫。

>[!TIP]
>
>無論您的組織何時必須保留創意資產的所有權，都可為團隊使用Creative Cloud Libraries。 某些範例包括：公司風格指南、共用專案和UI設計元素。
