# 建立適用於團隊的Creative Cloud Libraries

瞭解如何使用適用於團隊的Creative Cloud Libraries在群組或團隊內共用資產。 這些型別的程式庫並未繫結至個別使用者帳戶，而是屬於團隊。 您可以透過![建置影像](assets/Smock_Building_18_N.png)圖示來識別團隊資料庫。

>[!TIP]
>
>無論您的組織何時必須保留創意資產的所有權，都可為團隊使用Creative Cloud Libraries。 某些範例包括：公司風格指南、共用專案和UI設計元素。
