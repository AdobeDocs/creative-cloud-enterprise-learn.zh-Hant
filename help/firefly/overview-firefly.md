# [!DNL Firefly]Adobe概觀

![Firefly Hero Image](../assets/firefly.png)

Firefly 是一系列新系列的創意生成 AI 模型即將推出Adobe產品，主要著重于影像和文字效果的產生。 Firefly 提供新的方式來構想、創作和溝通，同時顯著改善創意工作流程。
