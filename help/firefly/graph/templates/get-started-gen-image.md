# 快速入門 — 產生影像

瞭解如何建立基本圖形：將一個提示節點變成一代節點變成一個輸出。 [開啟快速入門 — 產生影像範本](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:VA6C2:f556b557-d31b-4659-b9d1-358ed567c61d)。

[!BADGE 產業範例]{type=Informative tooltip=""}

* **零售** — 從簡報產生第一個產品主圖影像，以在接觸真正的行銷活動資產之前瞭解基本節點流程。
* **運作狀況** — 在縮放至完整內容行事曆之前，請先測試預留位置產品快照上最簡單的影像產生流程。
* **教育** — 在指派真正的專案工作之前，先建置第一個範例影像以示範新團隊成員的圖形。

>[!TIP]
>
>**開始之前** — 為獲得最佳結果，請根據您自己的品牌、產品和工作流程自訂此範本。 在使用任何輸出之前，交換參考影像、提示和複製。
