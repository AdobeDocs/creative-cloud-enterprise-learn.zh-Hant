# 字元模型產生

瞭解如何建立插圖的3D動畫樣式。 範本會產生一個基礎3D模型路徑，準備交給模型工具進行清除，而不是從空白場景開始。[開啟字元模型產生範本](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:US:4008fbc8-083b-5766-ace9-6abf998df840)。

[!BADGE 產業範例]{type=Informative tooltip="產業範例"}

* **戶外** — 從單一核准的字元摘要開始，建立用於封裝轉譯器和視訊的軌跡吉祥物字元的3D模型。
* **技術** — 從書面簡報產生基礎3D字元模型，準備交給模型工具進行清除。
* **教育** — 建置3D講師字元模型，用於課程的影片課程。

>[!TIP]
>
>**開始之前** — 為獲得最佳結果，請根據您自己的品牌、產品和工作流程自訂此範本。 在使用任何輸出之前，交換參考影像、提示和複製。
