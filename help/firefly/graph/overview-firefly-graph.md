# Adobe [!DNL Firefly Graph]總覽

如果您以前使用創作AI工具，您可能會習慣輸入提示、獲得一個結果，並在下次您需要類似工具時從頭開始。 Firefly Graph的運作方式不同。 您會建立圖形，而不是單一提示：視覺化的逐步工作流程，其中每個輸入、轉換和輸出都會連線並儲存在一起。

>[!TIP]
>
>把它想像成一個菜譜，而不是一頓飯。 建立圖表後，您可以將其關閉、在新的輸入上重新執行，或變更一個步驟並讓所有下游自動更新。

此頁面會逐步引導您瞭解建立第一個圖表所需的一切：核心概念、逐步第一個工作流程，以及可立即開啟和調整之現成範本的資料庫。
