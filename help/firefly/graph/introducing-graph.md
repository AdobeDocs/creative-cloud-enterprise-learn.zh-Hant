# 產生式填色

瞭解如何使用產生式填色，以使用簡單文字提示無損的方式從影像中新增、展開或移除內容。 創意填色可讓您增強創意並建立非凡的專案。 在[這裡](https://firefly.adobe.com/?locale=tw)試用Adobe Firefly。
