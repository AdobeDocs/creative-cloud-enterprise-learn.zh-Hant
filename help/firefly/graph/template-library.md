# &#x200B;5. 範本程式庫

Firefly Graph範本的快速參考索引，依每個範本產生或執行的內容組織。 每個範例都是起點 — 在生產中使用範本之前，交換您自己的品牌、產品和提示。

## 影像產生與樣式

| 圖表範本 | 說明 | 使用案例 |
|---|---|---|
| [**開始使用 — 產生影像**](/help/firefly/graph/templates/get-started-gen-image.md) | 將具有一個提示節點的基本圖表放入一個世代節點到一個輸出中。 | <ul><li>主圖影像</li><li>預留位置測試</li><li>影像範例</li></ul> |
| [**產生一致的字元**](/help/firefly/graph/templates/character-gen.md) | 載入一個字元的參考影像，然後交換每個新快照的場景或姿勢提示。 | <ul><li>循環的吉祥物</li><li>輻條字元</li><li>講師字元</li></ul> |
| [**樣式擷取**](/help/firefly/graph/templates/style-extraction.md) | 在參考影像中饋送，以擷取其顏色、光和紋理處理。 | <ul><li>外觀轉移</li><li>季節性相符專案</li><li>心情相符</li></ul> |
| [**日落視覺**](/help/firefly/graph/templates/sunset-vibes.md) | 從文字提示建立3D排版影像。 | <ul><li>標語覆蓋</li><li>季節性標語</li><li>Flash sale資產</li></ul> |
