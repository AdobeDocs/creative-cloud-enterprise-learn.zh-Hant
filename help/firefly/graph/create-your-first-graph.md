# &#x200B;3. 建立您的第一個圖表

一旦您知道節點、連線和範本是什麼，您就可以準備建立第一個工作流程。

1. **開啟空白畫布，或從範本開始。** 瀏覽節點程式庫或範本庫，尋找接近所需位置的起點。
2. **連線節點以建置您的工作流程。** 從節點的輸出到下一個節點的輸入繪製連線，以將步驟鏈結在一起 — 例如，輸入影像流入樣式節點，然後流入變數產生器，再進入匯出。
3. **為您稍後要變更的任何專案新增輸入控制項。** 將靜態值轉換為滑桿、檢色器或文字欄位，讓您（或隊友）可以調整關鍵引數，而不需編輯基礎圖形。
4. **執行圖表並預覽每個步驟。** 您可以在完整工作流程完成執行之前，於任何節點預覽輸出。
5. **變更一個輸入並重新執行。** 交換參考影像、調整樣式提示或調整變數 — 所有下游專案都會自動更新。 不會手動重建任何專案。

## 先嘗試什麼

如果您不確定從何處開始，請在範本庫中開啟其中一個現成的範本，而非空白畫布 — 這是從頭開始建立完整工作圖表之前，檢視完整工作圖表的最快方式。
