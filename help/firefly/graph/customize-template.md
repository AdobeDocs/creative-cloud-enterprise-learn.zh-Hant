# &#x200B;4. 自訂範本

每個範本都隨附範例輸入和一般使用案例。 它會變成只適用於&#x200B;*您*&#x200B;的工作流程，只有在它反映您自己的簡短、資產和結果時。

1. **將範例輸入取代為您自己的資產或簡報。**
2. **重新命名任何仍使用預留位置文字的節點或標籤。**
3. **在依賴結果之前，先執行一次圖表，完全執行。**
4. **變更一個變數並重新執行它**，以便您能夠檢視該步驟控制項。
5. **以您自己的名稱儲存您的自訂版本。**
