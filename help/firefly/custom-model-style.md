# 使用樣式建立自定義模型

瞭解如何在 Firefly 中建立「自定義模型」，以特定類型的樣式 （例如背景、顏色和色相） 為貴組織的品牌建立全新的影像。 「自定義模型」可讓您大規模建立相關的個人化內容。 請在這裡[&#128279;](https://firefly.adobe.com/)試Adobe Firefly。
