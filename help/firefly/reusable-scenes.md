# 為商品製作可重複使用的場景

瞭解如何為商品 （例如商店和配件） 建立可重複使用的場景。 建立可重複使用的場景可讓您按季或每次選擇來重新整理網站商品，從而加快數字銷售。 請在這裡[&#128279;](https://firefly.adobe.com/)試Adobe Firefly。
