# 使用樣式建立自定義模型

瞭解如何在Firefly中構建自定義模型，以使用特定類型的樣式（如背景、顏色和色調）為組織的品牌建立新的影像。 自定義模型允許您按比例建立相關的個性化內容。 嘗試Adobe Firefly[此處](https://firefly.adobe.com/)。
